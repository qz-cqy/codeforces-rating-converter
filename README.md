本项目是 CodeForces 新-旧规则转换计算器，帮助大家在新规则与旧规则的 rating 之间转换。

由于项目作者是中国人，暂时只有简体中文版本。如果你可以贡献更多语言，欢迎 Pull Request！

翻译规则：

- fork 仓库到本地，新建一个文件夹，用语言缩写命名（例如：en）；
- 拷贝一份代码，修改中文为您要翻译的语言；
- 翻译完毕后通过 Pull Request 提交修改，要写出翻译的语言。
- 不符合这个规范的翻译可能不被接受。

有其他错误？请先查看[项目 Wiki](https://github.com/qz-cqy/codeforces-rating-converter/wiki) 后进行反馈。
